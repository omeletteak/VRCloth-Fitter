# VRCloth Fitter

高度な衣装改変ツールのオープンソース代替品が必要であるという考えに基づき開発された、VRChatアバターに衣装を簡単にフィットさせるためのツールです。

## ロードマップ

このツールの目標は、アバターの衣装カスタマイズのための堅牢でコミュニティ主導の機能を提供することです。開発は以下のフェーズで計画されています。

### フェーズ1：基本的なスケールフィッティング

このフェーズでは、アバターのプロポーションに合わせて衣装のボーンスケールを調整することに焦点を当てます。

- [ ] **ボーンマッピングUI**:
    - [ ] アバターと衣装のボーンを名前の一致（例：`Hips`, `Chest`, `Spine`）に基づいて自動的に検出し、対応するペアをリスト化する。
    - [ ] ユーザーが手動でボーンの対応付けを編集・確定できるUIを提供する。
- [ ] **スケール計算**:
    - [ ] ボーンの「長さ」（子ボーンまでの距離）を計算する。
    - [ ] 周辺のメッシュ頂点を分析し、ボーン周りの「太さ」や「ボリューム」を推定する。
- [ ] **スケーリングの適用**:
    - [ ] 計算されたスケール調整を衣装ボーンの`localScale`に適用する。
    - [ ] 均一（Uniform）および非均一（Non-uniform）スケーリングのオプションを提供する。

### フェーズ2：高度なメッシュ変形（プロポーション改変）

これが中核機能であり、アバターの体型に合わせて衣装のメッシュを直接変形させることを目指します。

- [ ] **コントロールポイント（基準点）システム**:
    - [ ] ユーザーがアバターと衣装の両方のメッシュ上に対応する「アンカーポイント」を配置できるシステムを実装する。（例：胸のトップ、ウエストライン、肩先など）
- [ ] **メッシュ変形アルゴリズム**:
    - [ ] アンカーポイントに基づいてメッシュを変形させるアルゴリズムを開発する。まずは単純な距離に応じた加重平均法から着手する。
    - [ ] 将来的には、より滑らかな結果を得るためにラティス変形（Lattice Deformation）などの高度な技術を調査する。
- [ ] **非破壊的なワークフロー**:
    - [ ] すべての変更が非破壊的であることを保証し、元の衣装を上書きするのではなく、フィットさせた新しい衣装メッシュを生成するようにする。

### フェーズ3：機能拡張

- [ ] **ブレンドシェイプ（シェイプキー）同期**:
    - [ ] アバターのブレンドシェイプを衣装の対応するブレンドシェイプ（例：`Breasts_Big`）にリンクさせるUIを作成する。
    - [ ] Modular Avatarの`Blendshape Sync`コンポーネントのセットアップを自動化する。
- [ ] **マテリアル＆シェーダーユーティリティ**:
    - [ ] 衣装アイテムのマテリアルを、ユーザーが指定したシェーダー（例：lilToon）に一括で変換し、テクスチャの割り当てを試みるユーティリティ。

---

*このロードマップは、コミュニティからのフィードバックや開発の進捗に応じて変更される可能性があります。*
